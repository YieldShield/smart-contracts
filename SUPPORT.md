# Support

Use GitHub issues for reproducible bugs, documentation problems, and narrowly
scoped feature requests related to this repository.

Before opening an issue:

- Search existing issues and pull requests.
- Confirm you are using the current `main` branch or a clearly identified
  deployed version.
- Include the command, network, contract, or script involved.
- Include logs or reproduction steps when possible.

Do not open public issues for suspected vulnerabilities. Follow `SECURITY.md`
instead.

This repository is not a general help desk for wallet, RPC provider, exchange,
or unrelated application support.
